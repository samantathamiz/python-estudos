# Python Estudos

Esta pasta contém exercícios e projetos desenvolvidos para praticar conceitos básicos de Python. 
Cada subpasta corresponde a um tema ou conceito específico, permitindo que você siga a evolução dos estudos.


## Conteúdos das pastas

1. **Introdução (Hello World):** Primeiros códigos em Python, execução e boas práticas iniciais;
2. **Entrada e Saída de Dados:** Leitura de dados do usuário e exibição na tela;
3. **Tipos de Saída:** Formatação de saída, strings, números e concatenção;
4. **Graus e Ângulos (biblioteca math):** Conversão e cálculos com a biblioteca `math`; 
5. **Graus e Ângulos (deg2rad):** Conversão de ângulos usando `deg2rad` e funções trigonométricas; 
6. **Operadores:** Operadores aritméticos, lógicos e relacionais;
7. **Condicional:** Estruturas `if`, `elif` e `else`;
8. **Estruturas de Repetição:** Loops `for` e `while`; 
9. **Listas:** Criação, manipulação e iteração de listas; 
10. **Funções:** Criação e utilização de funções em Python;
11. **Dicionários:** Criação, acesso e manipulação de dados em dicionários.


## Objetivo

O objetivo desta pasta é **praticar e fixar os conceitos fundamentais de Python**, ajudando a compreender melhor a linguagem e ganhar segurança na programação.


## Tecnologias utilizadas

- Python 3;
- Biblioteca `math`.
